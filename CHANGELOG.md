# Changelog

## 1.1.0 - 2026-06-09

- 新增 AI eval 失败分析标签页，可按状态、错误原因、类别/套件、模型/代理聚合失败样本。
- 新增 Markdown 失败报告导出，适合用于 PR 评论、CI 摘要和回归分析记录。
- Added an AI eval failure drilldown tab with grouped failure summaries and examples.
- Added Markdown failure report export for pull requests, CI summaries, and regression notes.

## 1.0.0 - 2026-06-08

- Initial local release of JSONL Insight PWA.
- Added zero-build HTML/CSS/JS app shell.
- Added JSONL parser, field inference, filtering, numeric stats, category top values, date ranges, sampling, CSV export, Markdown export, and localStorage session recovery.
- Added offline PWA manifest and service worker.
- Added examples, documentation, CI, and Node standard library tests.
